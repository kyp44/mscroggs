\aproblem{1}{233}{\advent@xviii@i}{
  Define a qualifying sum as a sum written with only \mn{1} and \mn{2}.
  Let $f(n)$ be the number of ways to write $n$ as a qualifying sum.
  Each sum of $f(n)$ will end with either a $1$ or a $2$.
  In the former case, there are $f(n-1)$ ways to write the part preceding the $1$.
  In the latter case there are $f(n-2)$ ways to write the part preceding the $2$.
  Clearly then the total number of ways is the recursive relation
  \gath{
    f(n) = f(n-1) + f(n-2) \label{eqn:01:rec}
  }
  with the obvious base case $f(1) = 1$ and $f(2) = 2$ since $2$ can only be written as $1 + 1$ or simply $2$.
  This is the Fibonacci sequence but offset by one due to differing base cases.
  Since this has no nice closed form solution, the recursive function was evaluated in Python to get our answer $f(12) = 233$.
}
